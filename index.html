Built for WordPress

Like you, we love building awesome stuff with WordPress. Using template parts and many pre-built WordPress functions we have assemebled an easy to use framework.

For all skill levels

Espresso is designed to help people of all skill levels—designer or developer alike.

SEO

Developed from the begining to Work with the WordPress SEO Plugin by Yoast.

960 grid system

Espresso comes with 12, 16 and 24 columns, which can be used separately or together. The default grid is 24 but is easly changed in child themes.

Responsive design

Espresso include's Adapt.js setup for the 960 grid system. All you need to do is activate it in your child theme.

Multiple Layouts

Espresso has 6 Header layout options, 6 Content layout options and 7 Footer layout options. All layouts can be controlled via the theme admin area right within WordPress.
